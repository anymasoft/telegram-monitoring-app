# Telegram Monitoring App

**Веб-панель мониторинга Telegram-каналов и чатов.**

Next.js-приложение для отслеживания активности в выбранных Telegram-источниках: сбор сообщений, фильтры, дашборд. Развёртывается на Vercel.

---

## Возможности

- **Мониторинг каналов/чатов** по заданным критериям.
- **Дашборд** с фильтрацией и просмотром сообщений.
- **Готовность к Vercel** (App Router, serverless).

---

## Стек

Next.js (App Router) · React · Tailwind CSS · Vercel

---

## Быстрый старт

```bash
npm install
cp .env.example .env      # доступ к источнику данных Telegram
npm run dev
```
